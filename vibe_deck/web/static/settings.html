🛡️ Approval Level B

Y/N approval directly from the Stream Deck, no keyboard needed.

⚙️ Behavior

Auto-enter Approval Mode Automatically switch Deck to Y/N layout when agent needs approval (off = press blinking key to enter)

🎮 How it works

When an agent needs approval:

  1. Agent widget blinks 🛡️ "Approval?"
  2. Auto: Deck switches to Y/N layout automatically
    Manual: Press the blinking key to enter Y/N layout
  3. Rows 1-2 left half = 🟢 YES • right half = 🔴 NO
  4. Press any key in YES/NO zone to answer
  5. Deck restores normal layout automatically
