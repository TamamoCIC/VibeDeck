# Acknowledgments

VibeDeck would not exist without the incredible work of the open source community
who reverse-engineered the Elgato Stream Deck protocol — an undocumented HID
interface that Elgato does not officially support on Linux.

---

## Core Library

### python-elgato-streamdeck

**[Dean Camera (abcminiuser)](https://github.com/abcminiuser)** — The backbone of
this project. Dean reverse-engineered the Stream Deck HID protocol and built the
Python library that makes all of this possible. His meticulous documentation of
the protocol on [Elgato's docs site](https://docs.elgato.com/streamdeck/hid/)
is a gift to the open source community.

- GitHub: https://github.com/abcminiuser/python-elgato-streamdeck
- PyPI: https://pypi.org/project/streamdeck/
- License: MIT

## Hardware

### Elgato / Corsair

Thanks to [Elgato (a Corsair company)](https://www.elgato.com/) for creating the
Stream Deck hardware — a beautifully tactile piece of kit that inspires
creativity and productivity.

While Elgato does not provide an official Linux SDK, their published
[HID protocol documentation](https://docs.elgato.com/streamdeck/hid/) has been
invaluable to the community.
