## Agent skills

### Issue tracker

Issues live in GitHub Issues (`TamamoCIC/VibeDeck`), operated via the `gh` CLI. See `docs/agents/issue-tracker.md`.

### Triage labels

Default Matt Pocock skill labels: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context repo — one `CONTEXT.md` at root, `docs/adr/` for architectural decisions. See `docs/agents/domain.md`.
